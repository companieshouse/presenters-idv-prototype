What a companies house personal code is

You will need to use your Companies House personal code to connect your verified identity to our records. This is so that we know the correct person is linked to the name we have on file. The code is personal to you, not your company or a company you work for.

You'll need your personal code for various reasons. For example:

  • as part of some other company filings, such as the confirmation statement
  • when you are appointed as a director
  • if you become a person with significant control (PSC)

Your personal code is not the same as the company authentication code. If you have an authentication code, you should continue to keep this safe.

Only share the code with people you trust

You may need to share this code with people you trust to file on your behalf, or for your company. Keep this information secure, as you would with other unique codes such as your Unique Taxpayer Reference (UTR) for HMRC.

Identity verification makes it much more difficult for criminals to impersonate someone, but not impossible.
