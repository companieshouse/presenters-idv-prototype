Connecting existing accounts

If you have an existing WebFiling account, you can connect it to a GOV.UK One Login. This will allow you to sign in to WebFiling and other government services using the same account.

To connect the accounts:

  • Create or sign in to your GOV.UK One Login on the next screen.
  • We'll ask you to enter a security code that we'll send to your email address.

You'll only need to do this once, so that you can keep the existing information in your accounts.

! Warning You will lose any existing information if you do not use the same email address to sign in to each account. You must also use the same email address for your GOV.UK One Login.
